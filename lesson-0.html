What "Scala" means

The name is short for “Scalable Language.” Scala is designed to grow with you — short enough for a quick script, structured enough for a big system, and a vocabulary that gets more expressive as you reach for it.

What Scala values

A handful of recurring opinions surface again and again:

  • Expressions over statements. Many things you write evaluate to a value.
  • Types as design. They shape architecture as well as verify it.
  • Immutability by default. You opt into change when you need it.
  • Object-oriented and functional programming. Blends them seamlessly.
  • JVM compatibility. Java's libraries and ecosystem come along for free.

How to use these lessons

Each lesson is a list of small Scala snippets, most of them deliberately broken. The exercise is to look at the code, predict what it should do, try running it, tinker with the code, and figure out why it behaves that way and/or what it's missing. When you get stuck (or just whenever), try the Ask ChatGPT button which hands the original code snippet to AI that will try to explain what's going on. (It's usually pretty helpful but does occasionally miss the point or gives outright incorrect responses, so do some verification yourself!)
